# 政治概念地图

一个中文政治概念科普入口，用价值测评、议题地图和概念库帮助访客理解公共讨论中的价值取舍。

## 功能

- 40 道本地计算的价值测评题，生成五维结果和阅读线索。
- 五组价值坐标说明，帮助理解结果中的主要倾向和张力。
- 公共议题地图，把住房、教育、平台治理等现实问题连接到价值轴。
- 政治概念库，整理常见政治词条和容易混淆的概念对照。
- 测评路线，提供测评前、测评后和直接查内容三种阅读路径。

## 本地使用

这是一个单文件静态页面，不需要安装依赖或启动构建工具。

直接用浏览器打开 `index.html` 即可使用。

测评答案只在浏览器本地计算。页面不上传答案、不保存个人结果、不接入追踪代码。
